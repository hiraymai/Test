# 🎓 Онлайн Тест по Физической Культуре

Интерактивная система тестирования для студентов по физической культуре с поддержкой русского и казахского языков.

## ✨ Возможности

- 🌐 **Два языка**: Русский и Казахский
- 📝 **12 вариантов**: По 25 вопросов в каждом
- 👥 **Регистрация студентов**: ФИО, группа, дата
- 💾 **Сохранение результатов**: localStorage + экспорт CSV
- 📊 **Админ-панель**: Просмотр всех результатов
- 📱 **Адаптивный дизайн**: Работает на всех устройствах
- 🎯 **Тренажер**: Идеально для подготовки к экзаменам

## 🚀 Как запустить локально

```bash
# Установка зависимостей
npm install

# Запуск сервера
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

## 📋 Как использовать

### Для студентов:
1. Зарегистрируйтесь (ФИО, группа, дата)
2. Выберите язык (русский/казахский)
3. Выберите вариант (1-12)
4. Пройдите тест из 25 вопросов
5. Посмотрите свои результаты

### Для преподавателей:
1. Перейдите в админ-панель: [http://localhost:3000/admin](http://localhost:3000/admin)
2. Просмотрите все результаты студентов
3. Скачайте данные в CSV/Excel
4. Анализируйте статистику по группам

## 🎯 Темы тестов

- **Теория и методика физической культуры**
- **Медико-биологические основы спорта**
- **Педагогика в физическом воспитании**

## 📊 Структура вопросов

Каждый вариант содержит 25 вопросов:
- Теоретические основы физической культуры
- Анатомия и физиология спорта
- Педагогические принципы тренировок
- История физической культуры

## 💾 Хранение данных

Результаты сохраняются в:
- **localStorage** браузера (надежно и постоянно)
- **CSV экспорт** для анализа в Excel
- **Админ-панель** для просмотра статистики
